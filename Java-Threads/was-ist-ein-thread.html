Java Thread

Was ist ein Thread?

Java hat build in Multithreading

Mehrere Dinge gleichzeitig machen (-> Effizienz)

Jede "Einheit", die gleichzeitig läuft, heißt Thread

Wenn mehrere Threads gleichzeitig laufen heißt das Multithreading

Alle Threads nutzen den gleichen Address Space

(Process-Based Multitasking -> Jeder Prozess eigener Address Space)
