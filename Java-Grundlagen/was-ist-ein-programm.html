Was ist ein Programm?

Programme sind die Grundlage des Programmierens. Sie enthalten alle Anweisungen, die ein Computer ausführen soll.

📝Ein Programm ist eine Reihe von Anweisungen
👩‍💻Die ein Mensch verfassen kann
🖥️Und ein Computer verstehen kann
🔁Diese Abläufe werden in „Programmen" festgehalten
