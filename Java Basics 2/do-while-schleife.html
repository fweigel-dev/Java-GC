Die Do-While Schleife

In einer do While Schleife wird der Code in dieser mindestens einmal ausgeführt danach wieder nur noch durch die zutreffende Bedingung.

Hier wird Z solange erhöht bis es 10 erreicht hat und wird mindestens einmal ausgegeben.
