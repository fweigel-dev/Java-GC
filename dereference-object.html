Wie man ein Objekt in Java dereferenziert

1. Die Referenz gleich `null` setzen

MyClass obj = new MyClass();
obj = null;

Durch das Setzen der Referenz auf null wird das Objekt für den Garbage Collector freigegeben, sofern keine anderen Referenzen darauf existieren.

2. Die Referenz mit einer anderen überschreiben

MyClass obj1 = new MyClass();
MyClass obj2 = new MyClass();
obj1 = obj2;

Das ursprüngliche Objekt, auf das obj1 zeigte, wird dereferenziert und kann vom Garbage Collector entfernt werden, wenn keine anderen Referenzen darauf existieren.

3. Ein anonymes Objekt verwenden

myMethod(new MyClass());

Das erstellte Objekt wird nach dem Methodenaufruf automatisch dereferenziert, da keine bleibende Referenz darauf existiert.
