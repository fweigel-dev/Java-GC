Vererbung

Grundkonzept

In Java ist es möglich, Attribute und Methoden von einer Klasse an eine andere zu vererben. Wir unterteilen das „Vererbungskonzept“ in zwei Kategorien:

Subclass (Child/ Kind/ Unterklasse) → Klasse die von einer anderen Erbt

Superclass (Parent/ Eltern/ Oberklasse) → Klasse von der vererbt wird



Um von einer Klasse zu erben, verwenden Sie das Schlüsselwort extends.


Es ist nützlich für die Wiederverwendbarkeit von Code: Wiederverwendung von Attributen und Methoden einer bestehenden Klasse, wenn Sie eine neue Klasse erstellen.
